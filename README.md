# 📊 Sales Data Analysis using SQL

## 📌 Project Overview

This project focuses on analyzing sales data using SQL to uncover valuable business insights. The analysis includes customer purchasing behavior, product performance, revenue trends, and sales metrics using SQL queries.

## 🎯 Objectives

- Analyze sales performance
- Identify top-performing products
- Understand customer purchasing patterns
- Calculate revenue and sales metrics
- Generate business insights using SQL

## 🛠️ Tools & Technologies

- MySQL
- SQL
- Excel
- Git
- GitHub

## 📂 Dataset

The project uses a car sales dataset containing sales transactions and customer information for SQL analysis.

## 🔍 SQL Concepts Used

- SELECT Statements
- WHERE Clause
- ORDER BY
- GROUP BY
- HAVING
- Aggregate Functions (SUM, COUNT, AVG, MAX, MIN)
- Aliases
- Filtering
- Data Sorting

## 📈 Key Insights

- Identified top-selling products.
- Analyzed revenue generated from sales.
- Examined customer purchasing trends.
- Generated department-wise and product-wise sales reports.
- Applied SQL queries to answer business-related questions.

## 📁 Project Files

- `sales_analysis.sql` – SQL queries used for analysis
- `car_prices.csv` – Dataset used for the project

## 🚀 Learning Outcomes

This project strengthened my understanding of SQL by working with real-world sales data. It improved my ability to write efficient SQL queries, analyze datasets, and generate meaningful business insights.
